# pydoc -p portnum_xxxx   



pydoc的使用方法(转)[@more@]
很简单的。运行一下pydoc看看？
会发现它有一个选项为-p
即运行在http协议上。
我们这样运行一下：pydoc -p 7766
然后打开你的浏览器输入：http://localhost:7766,呵呵，你看到了什么？
